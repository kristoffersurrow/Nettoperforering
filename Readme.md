Nettoperforering.dk er en hjemmeside lavet i Wordpress CMS, for smedefirmaet Netto Perforering.

Da jeg ikke længere har adgang til at redigere sitet, og da den færdige side endnu ikke er publiseret,
har jeg:
- skrevet et par af siderne i rå HTML og CSS
- lavet en video af "Mønsterguiden"
- linket til den oprindelige "Mønsterguide" på nettoperforering.dk's ufærdige side (som ikke er garanteret at virke)

Det har jeg lært:
- JavaScript
- JQuery
- AJAX
- PHP
- HTML
- CSS
- Bootstrap
